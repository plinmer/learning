# 数据库分类
- 关系型数据库(SQL)
    - MySQL
    - SqlServer
    - Oracle
    - PostgreSQL
    - SQLite 
- 非关系型数据库(NOSQL)
    - KV数据库
        - Redis
        - Memcached

        - HBase
        - ClickHouse
    - 面向文档数据库
        - MongDB
        - CouchDB
    - 搜索引擎数据库
        - Elasticsearch
